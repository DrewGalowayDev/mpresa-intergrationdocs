Carbon Credit Platform API Docs

Introduction
API for registering farmers, estimating carbon reductions, and issuing tokenized carbon credits on ICP blockchain.
Welcome to the Carbon Credit Platform. This documentation describes endpoints to manage farmer onboarding, carbon MRV (Monitoring, Reporting & Verification), and smart contract interactions for minting and tracking carbon tokens.
Register Farmer
Enroll new farmer into the program.
POST/v1/farmers/register
Request Parameters
ParameterTypeDescriptionRequired
namestringFarmer's full nameYes
locationstringGPS coordinatesYes
farmSizenumberArea in hectaresYes
practicesarrayAdopted practices listYes
Example Request
{
  "name": "Jane Doe",
  "location": "-1.2921,36.8219",
  "farmSize": 5.5,
  "practices": ["no-till","agroforestry"]
}
Example Response
{
  "status":"success",
  "farmerId":"FARM12345"
}

Issue Tokens
Mint carbon credits to farmer's ICP wallet.
POST/v1/credits/issue
Request Parameters
ParameterTypeDescriptionRequired
farmerIdstringFARM IDYes
amountnumberTokens to mintYes
walletAddressstringICP walletYes
Example Request
{ "farmerId":"FARM12345","amount":12.3,"walletAddress":"abcd-efgh-ijkl" }
Example Response
{ "status":"success","transactionId":"TXN67890" }
